# SkillSentinel

SkillSentinel is an AI-based system that predicts skill obsolescence using job market trends and provides personalized micro-learning recommendations with verification.

## Problem
People often learn skills without knowing whether those skills will remain relevant in the future.

## Solution
This project analyzes job market trends and user profiles to identify at-risk skills and recommend targeted improvements.

## Status
Project setup completed. Development in progress.
